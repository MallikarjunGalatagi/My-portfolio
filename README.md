Welcome to the repository of my personal developer portfolio. This is a premium, modern, and fully animated single-page application built to showcase my technical skills, academic accomplishments, features, and full-stack project showcases.

Designed with an **Apple-inspired Glassmorphism UI** and **Code Slate (Space Dark)** aesthetics, the website offers a high-performance, responsive layout optimized for all modern desktop and mobile viewports.

---

## 🎨 Design Features

- **🌓 Dynamic Dual Themes**: 
  - **Code Slate (Default)**: A beautiful space-dark coding canvas featuring CSS animated drifting gradient light blobs (aurora meshes) and soft glows.
  - **Evernote**: A warm, clean, editorial cream paper aesthetic with solid white card layouts and branding-aligned emerald green highlights.
- **🌫️ Glassmorphic Cards**: Features custom card wrappers with variable white opacity layers, `18px` backdrop-blur filters, thin borders, drop-glow outlines on hover, and diagonal sweeps of linear reflection shine (`sweep-shine`).
- **🔗 Responsive Timeline**: An interactive vertical Education timeline with custom connector lines that dynamically scale to fit Bootstrap grids on all viewports.
- **⚡ Mobile Scroll Optimization**: Automatically disables heavy GPU operations (backdrop filters and 3D gyroscope tilts) on screen widths under `768px` to ensure a buttery-smooth 60fps scrolling experience.
- **📬 Native Redirection Channels**: Pre-configured email mailto strings and direct call links for smooth communication channels.

---

## 🛠️ Technology Stack

- **Frontend**: HTML5, Vanilla CSS3 (Custom Grid, Flexbox, Variable Tokens), JavaScript (ES6+)
- **Libraries & CDNs**:
  - [Bootstrap 5](https://getbootstrap.com/) (Responsive Grid Structure)
  - [AOS (Animate On Scroll)](https://michalsnik.github.io/aos/) (Smooth entry transitions)
  - [Typed.js](https://github.com/mattboldt/typed.js/) (Dynamic header subtitles)
  - [Vanilla-Tilt.js](https://micku7zu.github.io/vanilla-tilt.js/) (3D parallax hover card effects)
  - [Font Awesome 6](https://fontawesome.com/) (Icon sets)
- **Deployment**: GitHub Pages

## 💻 Local Setup and Development

To run this project locally on your machine, you do not need any compilation or build dependencies. Simply clone the repository and run a local server:

1. **Clone the repository**:
   ```bash
   git clone https://github.com/MallikarjunGalatagi/My-portfolio.git
   cd My-portfolio
   ```

2. **Start a local development server**:
   * Using **Python 3**:
     ```bash
     python -m http.server 8080
     ```
   * Using **Node.js (http-server)**:
     ```bash
     npx http-server -p 8080
     ```

3. **Open in browser**:
   Navigate to [http://127.0.0.1:8080](http://127.0.0.1:8080) to test updates instantly.
